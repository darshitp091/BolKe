## 🚀 Overview
**BolKe** is a revolutionary platform designed to empower millions of Indian small business owners ("Vyaaparis") who may not be tech-savvy. With BolKe, creating a professional website is as simple as sending a voice note. Just speak about your business in your native language, and our AI builds a premium, WhatsApp-ready storefront in seconds.

## ✨ Core Features (Hackathon Prototype)
- **🎙️ Voice-First Generation**: Real-time keyword detection extracted from speech to build niche-specific sites.
- **⚡ Multi-Niche Engine**: Specialized templates for **Clothing, Mechanics, Healthcare, Jewellery, and Manufacturing**.
- **💬 WhatsApp Commerce**: One-click "Order on WhatsApp" integration to convert visitors into customers.
- **🎨 Premium Designs**: Modern, high-conversion aesthetics with smooth animations and dynamic theming.
- **🤖 100% Local AI Support**: Infrastructure ready for Ollama (Llama 3/DeepSeek) for zero cost, private processing.

## 🛠️ Tech Stack
- **Frontend**: React (Vite), Framer Motion, Tailwind CSS, Lucide Icons.
- **Backend**: Node.js (Express), SQLite (Local Data), Multer (Audio).
- **AI Stack**: Ollama (Llama-3-8B), Whisper (Local STT), Groq (Cloud Fallback).

## 📦 Run Locally

1. **Clone & Install**:
   ```bash
   git clone https://github.com/darshitp091/BolKe.git
   cd BolKe
   npm install
   ```

2. **Setup AI (Optional for Mock Mode)**:
   Follow the [SETUP_LOCAL_AI.md](./SETUP_LOCAL_AI.md) to install Ollama.

3. **Launch**:
   ```bash
   npm run dev
   ```

## 🌍 Deployment
This project is optimized for **Vercel**. 
1. Connect your repository to Vercel.
2. The `vercel.json` will handle all routing automatically.
3. For the hackathon demo, the site will run in **High-Res Mock Mode** if no backend is detected.
